# 비전아이티 업무관리시스템 (ERP) 구현 현황

## 1. 파일 구조

### 1.1 HTML 파일
- index.html: 메인 대시보드 페이지
- dashboard-summary.html: 사업계획 진행 관리 페이지
- html/department-management.html: 부서 관리 페이지
- html/employee-management.html: 사원 관리 페이지
- html/user-management.html: 사용자 관리 페이지
- html/login.html: 로그인 페이지
- html/register.html: 회원가입 페이지

### 1.2 JavaScript 파일
- js/script.js: 공통 기능 스크립트 (로그인 상태 확인, 사용자 정보 표시 등)
- js/dashboard-summary.js: 사업계획 진행 관리 기능
- js/department-management.js: 부서 관리 기능
- js/employee-management.js: 사원 관리 기능
- js/user-management.js: 사용자 관리 기능
- js/login.js: 로그인 기능
- js/register.js: 회원가입 기능

### 1.3 CSS 파일
- css/styles.css: 전체 스타일시트

### 1.4 서버 관련 파일
- server.js: 메인 서버 파일
- models/users.js: 사용자 데이터 모델
- routes/user.js: 사용자 관련 API 라우트

### 1.5 기타 파일
- package.json: 프로젝트 설정 및 의존성
- erp_system_requirements.txt: 시스템 요구사항

## 2. 구현된 주요 기능

### 2.1 사용자 인증 및 관리
- 로그인/로그아웃 기능
- 사용자 등록 및 관리
- 권한 관리

### 2.2 부서 관리
- 부서 목록 조회
- 부서 추가/수정/삭제
- 조직도 시각화
- 부서장 지정
- 부서 구성원 관리

### 2.3 사원 관리
- 사원 정보 등록 및 관리
- 인사 정보 관리 (경력, 학력, 자격증, 가족 정보 등)
- 사원 검색 기능

### 2.4 사업계획 진행 관리 (대시보드)
- 사업계획 목록 조회
- 사업계획 등록/수정/상세 조회
- 진행 상태 관리 및 시각화
- 마일스톤 및 이슈 관리
- 필터링 및 검색 기능
