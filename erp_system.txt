# ERP 시스템 구현 현황 (2025-10-27)

## 구현된 페이지 및 기능 목록

### 1. index.html (메인 대시보드)
- 기본적인 대시보드 UI 구현
- 사이드바 메뉴 구조 구현
- 로그인 상태 확인 기능

### 2. html/employee-management.html (사원 관리)
- 사원 목록 표시 및 검색 기능
- 사원 상세 정보 표시 및 편집 기능
- 탭 기반 UI로 다음 정보 관리:
  - 기본 정보 (이름, 주민번호, 연락처, 이메일, 부서, 직급, 입사일, 재직상태, 주소)
  - 부양가족 정보 관리 (추가/수정/삭제)
  - 학력 정보 관리 (추가/수정/삭제)
  - 자격증 정보 관리 (추가/수정/삭제)
  - 경력 정보 관리 (추가/수정/삭제)
  - 은행 계좌 정보 관리
- 신규 사원 추가 기능
- 사원 정보 저장 및 삭제 기능
- 경력증명서 출력 기능 (UI만 구현)
- 모달 기반의 정보 추가 기능

### 3. html/department-management.html (부서 관리)
- 부서 목록 표시 및 검색 기능
- 부서 상세 정보 표시 및 편집 기능
  - 부서 코드, 부서명, 부서장, 상위 부서, 상태, 생성일, 부서 설명
- 부서 구성원 목록 표시
- 조직도 시각화 기능
- 신규 부서 추가 기능
- 부서 정보 저장 및 삭제 기능
- 부서장 선택을 위한 모달 기능

### 4. html/user-management.html (사용자 관리)
- 사용자 목록 표시 및 검색/필터링 기능
- 사용자 추가 모달
  - 이름, 사용자명, 이메일, 전화번호, 부서, 직급, 권한, 상태, 비밀번호 설정
- 사용자 편집 모달
  - 기존 사용자 정보 수정 기능
- 비밀번호 초기화 모달
- 사용자 삭제 모달
- 사용자 일괄 관리 기능 (활성화/비활성화)
- 사용자 정보 내보내기 기능 (Excel, PDF - UI만 구현)

### 1. js/script.js

### 2. js/employee-management.js
- 사원 관리 페이지 관련 기능 구현
- 테스트 데이터 로드 함수
- 사원 정보 표시 및 편집 기능
- 부양가족, 학력, 자격증, 경력 정보 관리 기능

### 3. js/department-management.js
- 부서 관리 페이지 관련 기능 구현
- 부서 목록 및 상세 정보 표시
- 부서 구성원 표시
- 조직도 렌더링 기능
- 부서장 선택 기능

### 4. js/user-management.js
- 사용자 관리 페이지 관련 기능 구현
- 사용자 목록 표시 및 관리
- 사용자 추가/편집/삭제 기능
- 비밀번호 관리 기능

## CSS 스타일

### css/styles.css
- 전체 애플리케이션의 공통 스타일
- 사이드바 스타일
- 메인 컨텐츠 영역 스타일
- 테이블 및 폼 스타일
- 조직도 시각화를 위한 스타일

## 데이터 관리
- 현재는 클라이언트 측 테스트 데이터 사용 (하드코딩된 JavaScript 객체)
- 실제 백엔드 API 연동은 아직 구현되지 않음

## 구현된 기능의 특징
- 모든 페이지는 완전한 반응형 디자인으로 구현
- Bootstrap 5.2.3 프레임워크 사용
- Font Awesome 6.4.0 아이콘 사용
- 모달 기반의 데이터 입력 및 편집 인터페이스
- 테이블 기반의 데이터 표시
- 탭 기반의 복잡한 데이터 구조 관리

## 미구현 기능
- 실제 백엔드 API 연동
- 데이터 유효성 검증
- 파일 업로드 및 관리
- 실제 사용자 인증 및 권한 관리
- 회계, 영업, 계약, 유지보수, 사업계획 관련 페이지
