# 비전아이티 업무관리시스템(ERP) 구현 현황 요약

## 1. 구현된 파일 구조

### HTML 파일
- index.html: 메인 대시보드
- dashboard-summary.html: 사업계획 진행 관리
- html/dashboard-analytics.html: 영업 진행관리
- html/department-management.html: 부서 관리
- html/employee-management.html: 사원 관리
- html/user-management.html: 사용자 관리
- html/login.html: 로그인
- html/register.html: 회원가입
- html/business-plan.html: 사업계획 관리
- html/company-document-management.html: 회사 문서 관리

### JavaScript 파일
- js/script.js: 공통 기능 스크립트
- js/dashboard-summary.js: 사업계획 진행 관리 기능
- js/dashboard-analytics.js: 영업 진행관리 기능
- js/department-management.js: 부서 관리 기능
- js/employee-management.js: 사원 관리 기능
- js/user-management.js: 사용자 관리 기능
- js/login.js: 로그인 기능
- js/register.js: 회원가입 기능
- js/business-plan.js: 사업계획 관리 기능
- js/company-document-management.js: 회사 문서 관리 기능

### CSS 및 서버 파일
- css/styles.css: 전체 스타일시트
- server.js: 메인 서버 파일
- models/users.js: 사용자 데이터 모델
- routes/user.js: 사용자 관련 API 라우트

## 2. 구현 완료된 기능

### 사용자 인증 및 관리
- 로그인/로그아웃 기능
- 사용자 등록 및 관리
- 권한 관리

### 부서 관리
- 부서 목록 조회
- 부서 추가/수정/삭제
- 조직도 시각화
- 부서장 지정
- 부서 구성원 관리

### 사원 관리
- 사원 정보 등록 및 관리
- 인사 정보 관리 (경력, 학력, 자격증, 가족 정보 등)
- 사원 검색 기능

### 사업계획 진행 관리
- 사업계획 목록 조회
- 사업계획 등록/수정/상세 조회
- 진행 상태 관리 및 시각화
- 마일스톤 및 이슈 관리
- 필터링 및 검색 기능

### 사업계획 관리
- 사업계획 목록 조회 및 필터링
- 사업계획 상세 정보 조회
- 사업계획 등록/수정
- 사업계획 승인/반려 처리
- 목표 및 성과지표 관리
- 활동 계획 관리
- 예산 계획 관리
- 첨부 파일 관리
- Excel 내보내기 기능

### 영업 진행관리
- 영업 현황 요약 통계 (총 영업기회, 계약 건수, 목표 달성률, 예상 매출)
- 영업 진행 현황 목록 조회 및 필터링
- 영업 상세 정보 조회
- 영업 등록/수정
- 영업 활동 추가 및 관리
- 단계별 영업 현황 시각화
- 담당자별 영업 실적 분석
- 월별 영업 실적 추이 분석
- Excel 내보내기 기능

### 회사 문서 관리
- 문서 목록 조회 및 관리
- 문서 유형별 필터링 (계약서, 증명서, 보고서, 재무 문서, 규정/지침, 회의록 등)
- 문서 검색 기능
- 문서 등록/수정/삭제 기능
- 문서 상세 정보 조회
- 문서 이력 관리
- 보존기한 및 보안등급 관리
- 관련 부서 지정 기능
- 문서 파일 업로드 및 다운로드
- 문서 목록 내보내기 기능
